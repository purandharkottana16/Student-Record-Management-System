# Student-Record-Management-System
This is a simple Student Record Management System built using the Flask framework. It allows users to add, view, and delete student records through a clean and attractive web interface.

🚀 Features ➕ Add new student details 📋 View all student records Fast and light weight flask

🛠️ Technologies Used Python Flask HTML CSS
